# Assignment 4 — Deep Learning for Image Classification

## Overview

This assignment explores deep learning approaches to image classification using Convolutional Neural Networks (CNNs) built with Keras/TensorFlow.

## Topics Covered

- **CNN Architecture Design** — Convolutional, pooling, and fully connected layers
- **Training & Evaluation** — Loss curves, accuracy metrics, confusion matrices
- **Transfer Learning** — Leveraging pre-trained models
- **Data Augmentation** — Improving generalization through image transformations
